# Twitter Sentiment Analysis Using Machine Learning

This project performs **Sentiment Analysis** on Twitter data using **Natural Language Processing (NLP)** and **Machine Learning** techniques.

The project cleans tweet text, converts it into numerical features using TF-IDF Vectorization, and classifies sentiments using:

- Naive Bayes
- Support Vector Machine (SVM)

---

## 📌 Features

- Text Cleaning & Preprocessing
- Stopword Removal
- Lemmatization
- TF-IDF Vectorization
- Sentiment Classification
- Naive Bayes Model
- Support Vector Machine (SVM) Model
- Accuracy Evaluation
- Confusion Matrix Visualization
- Sentiment Distribution Plot

---

## 📂 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK

---

## 📊 Dataset

Dataset Used:

`Twitter_Data.csv`

The dataset contains Twitter text data and sentiment labels.

Possible sentiment categories may include:
- Positive
- Negative
- Neutral
